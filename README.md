
- Slide 1-3: A Transição Crítica (RN vs RF)

## 🎓 COMPETÊNCIAS DESENVOLVIDAS

Ao final do Encontro 4, o aluno será capaz de:

1. ✅ **Diferenciar RN e RF** com exemplos de 3 domínios
2. ✅ **Escrever RF profissional** usando anatomia [Ator]+[Dever]+[Ação]+[Objeto]+[Condição]
3. ✅ **Reconhecer RF testável** vs ambíguo/vago
4. ✅ **Categorizar RF** em D/P/I/S com justificativa
5. ✅ **Priorizar com MoSCoW** argumentando MUST vs SHOULD
6. ✅ **Rastrear RF a RN** eliminando órfãos
7. ✅ **Identificar conflitos** entre requisitos
8. ✅ **Criar artefato profissional** (planilha de RF) pronto para entrega

**Avaliação:** Planilha final deve passar em ≥70% dos 8 critérios (Slide 20)

## 🆘 TROUBLESHOOTING

### Problema: Arquivo HTML não abre no navegador
**Solução:** 
1. Duplo-clique não funciona? Abra navegador manualmente
2. Vá para File → Open → Selecione `apresentacao-rf.html`
3. Se ainda não funciona, use caminho completo: `file:///C:/Users/...`

### Problema: Download PDF não funciona
**Solução:**
1. Verifique conexão internet
2. Teste em outro navegador (Chrome é mais confiável)
3. Se erro persistir, use Print → Salvar como PDF (Ctrl+P)

### Problema: Alunos não entendem categorização D/P/I/S
**Solução:**
1. Use dinâmica 3.1 mais extensamente (Sorteio de Categorias)
2. Crie exemplos com projeto que alunos conhecem (Instagram, WhatsApp)
3. Mostre fluxo completo: D (dados do perfil) → P (recomendação) → I (feed) → S (notificação)

### Problema: Laboratório ultrapassou tempo
**Solução:**
1. Corte Dinâmica 4.2 (Debate) se necessário
2. Deixe laboratório para próxima aula (continuação)
3. Nada impede apresentar slides + dinâmicas dinâmicas em 2 encontros
